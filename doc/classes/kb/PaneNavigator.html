    Class: kb.PaneNavigator

    Overview

    Note: If using Knockout, 'hasHistory', 'activePage', and 'activeUrl' methods can be can be observed for changes.

    Pane Navigator that simply replaces the active page element when it changes and cleans up the previous if it exists.

    Constructor Details

    - (void) constructor(el, options)

    Parameters:

    • el (Element) — the container element for the pane navigator
    • options (Object) — create options

    Options Hash: (options):

    • no_remove (Boolean) — do not remove elements from the DOM with they are inactive. Useful if you are using a static DOM hierarchy rather than generating the elements dynamically.
    • no_history (Boolean) — if you would like to not store the history of panes, but only have one in memory at a time (default is with history)
    • transition (String|Object) — default transition options. Either a name or {name: 'TransitionName', option1: option2: ...}
